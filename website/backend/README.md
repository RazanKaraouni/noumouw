# Noumouw API

Node.js / Express backend for the Nomou platform.

## Run

```bash
npm install
cp .env.example .env   # fill in secrets locally
npm run dev
```

## Tests

```bash
npm test
```

## Dev utilities

One-off migration and maintenance scripts live in the repository root [`dev-tools/`](../../dev-tools/) directory. **Dev utilities in /dev-tools are never deployed to production.**
